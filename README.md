BellCorp Event Management Application is a full-stack MERN application designed to provide a seamless Event Discovery and Registration Experience.

The platform allows users to browse a large collection of events, search and filter them dynamically, and manage their event registrations securely through authentication-protected workflows.

The application is built with scalability and usability in mind, handling continuously growing event data while maintaining smooth navigation and state across the user journey.
